# TopTalk
****
### Getting Started
Get API Keys
   + [DiffBot](https://www.diffbot.com/pricing/)
   + [NewsAPI](https://newsapi.org/register)

Run the following: 
```
cd src && npm install;
touch .env: 
```
`.env` format:
```
API_KEY=<NEWSAPIKEY>
DIFFBOT_TOKEN=<DIFFBOTTOKEN>
PORT=3333
```

### Run TopTalk
```
npm run start
```
